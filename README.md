# Облако тэгов

Приложение для построения облака тэгов по переданному файлу, с графическим интерфейсом.
При генерации облака можно настраивать:
  * Форму облака
  * Шрифт слов
  * Цвет слов
  * Размер изображения
  * Фильтрация исходного текста(словарь запрещенных слов, определенные части речи)
